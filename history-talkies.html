ဟိုး..ရှေးရှေးတုန်းက သမိုင်းဝင် တော်ကီများ စုစည်းမှု

Classic parody dialogue collection

ငါ မရှိတော့ရင် ငါ့မိဖုရားကို သူ သိမ်းပိုက်တော့မှာ ငါ.. စိတ်မချဘူး ဟဲ့.. ငယ်ကျွန် စောဥမ္မာကို သွားသတ်ချေ
( သတိုးမင်းဖျား )
မောင်တော်ရယ်.. ကိုယ်လည်း ခဏတဖြုတ်ပဲ တတ်နိုင်တာကို ကိုယ်မကုန်တာ သူများ.. စားပါစေလား.. ကုသိုလ်တောင် ရသေး
( စောဥမ္မာ )
ခုနစ်ပြည်ထောင်ကို စစ်မက်ခေါ်ရတာ သူတို့တိုင်းပြည်ကို မက်လို့မဟုတ်ဘူး ပပကြောင့်
( မင်းကုသ )
တစ်ကောင်မဟုတ် တစ်ကောင်ရမှာပါ အရေးမကြီးပါဘူး.. ဘယ်ကောင်နဲ့ရရ မိဖုရားပါပဲ တစ်ကောင်ကို တစ်ကောင်သတ်ကြရင်း နောက်ဆုံး.. နှာဗူးခြောက်ကောင် လျော့သွားတာပေါ့
( ပပဝတီ )
အချစ်ရေးမှာ အသက်ကွာတာ အဓိက မဟုတ်ပါဘူး ဓာတုကလျာ
( နတ်ရှင်နောင် )
အသက်ငယ်တော့ ပိုအားသုံးနိုင်တာပေါ့ကွယ် ဖွင့်မမီတောင် ပိတ်ပွဲမီရမယ်ဆိုတဲ့ ငါ့မောင်စိတ်ဓာတ်တော့ ကြိုက်တယ်.. သက်လား
( ဓာတုကလျာ )
ဓား၊ လှံ၊ သံလျက် ဘယ်လက်နက်နဲ့မှ မသေလိုပါ အရှင်မင်းကြီး၏ စောဥမ္မာသာ လိုပါသည်
( ငတက်ပြား )
ငတက်ပြားဆိုတဲ့ကောင် အနံ့နဲ့တင် သေသွားစေရမယ် မောင်တော်
( စောဥမ္မာ )
မအိုဇာကို ပြောပေးပါ ကွမ်းတရာ ရေတစ်မှုတ်နဲ့ နှုတ်ဆက်ပေးပါလို့
( ဗျတ်ဝိ )
မသာ.. လူကို ကွမ်းယာသည်များ မှတ်နေလား
( မအိုဇာ )
မွှေးနွန်းရယ်.. မောင်တော် ဘုန်းကံနိမ့်ရင် အသက်အန္တရာယ် ရှိပေမယ့် မောင်တော်ရဲ့ ဘုန်းကံထက် မွှေးနွန်းကို တန်ဖိုးထားပါတယ်
( မင်းနန္ဒာ )
မောင်တော်ရယ်.. ထဘီလျှော်ရုံနဲ့ ဘုန်းကံ နိမ်းကြေးလား.. ဟတ်. ဟတ်
( မွေးနွန်း )
သားတော် အမိန့်ကို နာခံခဲ့ပေမယ့် သားတော်ရဲ့ နှလုံးသားက ခမည်းတော်အမိန့်ကို နာခံဖို့ ငြင်းဆန်ခဲ့ပါတယ်
( ကျန်စစ်သား )
ငါ့စော်ကို ကျော်တဲ့ မသာကများ.. လိ့
( အနော်ရထာ )
သိပ်ချစ်တက်တဲ့ ကျွန်တော့်နှလုံးသားကို သံကွန်ချာ ခုနစ်ထပ်အုပ်ပြီး အမိန့်ပေးလို့မရဘူး ဆရာတွတ်
( ဗိုက်ကလေး )
မိန်းမ ရမှ မှားမယ်မှန်း သိမယ် တပည့်ကြီး
( ဆရာတွတ် )
မင်းမရှိတဲ့ ဘဝမှာ ငါတစ်ယောက်တည်းများ ရှင်သန်နေမယ်ထင်လား
( ရိုမီယို )
မွှန်နေတယ်ပေါ့ ဟတ်.. ဟတ်
( ဂျူးလိယက် )
စုစုရယ် ခုနစ်နှစ် ခုနစ်မိုးက မနေ့တနေ့ကလို ပါပဲလား
( သီပေါမင်း )
ခဏတဖြုတ်ကိုး.. ဒါတောင် ဟိုကောင်မ ဒိုင်းခင်ခင်နဲ့ နန့်ကြောဆွဲသေး
( စုဖုရားလတ် )
